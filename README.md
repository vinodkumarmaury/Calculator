# Calculator App

This is a simple calculator web application built using HTML, CSS, and JavaScript. It provides basic arithmetic operations such as addition, subtraction, multiplication, and division.

# Feature
• Addition (+)

• Subtraction (-)

• Multiplication (*)

• Division (/)

• Clear (AC)

• Clear Entry (CE)

• Toggle Sign (+/-)
